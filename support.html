Before Contacting Support

Please include as much of the following as you can:

  • PromptMute version.
  • macOS version and device type.
  • Whether you are using the browser build or desktop app.
  • What you expected to happen.
  • What actually happened.
  • Whether the issue happens with the built-in sample text.

Do not send raw API keys, passwords, private customer data, or mapping exports unless we explicitly ask for a sanitized sample.

Troubleshooting

Copy or paste does not work

  • Check macOS or browser clipboard permissions.
  • Try the desktop app if the browser blocks clipboard access.
  • Use Export as a temporary fallback.

Cleaned Prompt is empty

  • Confirm Original Text has content.
  • Run Scan after changing large text.
  • Click Clean & Copy to generate the output.

Restore does not replace placeholders

  • Confirm Mapping has active entries.
  • Do not clear session before restoring AI replies.
  • Unlock saved mapping with the correct passphrase if needed.

macOS blocks the app

  • Use a signed and notarized DMG from the official download page.
  • Confirm the app name is PromptMute.
  • Contact support with the exact Gatekeeper message.

Security Reports

If you believe you found a security issue, contact support@promptmute.app with a clear description and reproduction steps.

  • Do not include real secrets in the initial report.
  • Use a minimal sanitized example where possible.
  • Include the app version and operating system.
  • We may ask for additional details after triage.
